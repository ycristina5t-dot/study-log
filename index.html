🔑 GitHub連携の設定

データをGitHubに保存することで、スマホ・PCどちらからでも同じデータを見られます。

  1. GitHubにログイン
  2. Settings → Developer settings → Personal access tokens → Tokens (classic)
  3. Generate new token (classic) をクリック
  4. スコープで repo にチェック
  5. 生成されたトークンをコピーして下記に貼り付け
